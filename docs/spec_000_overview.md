# spec_000_overview

## Purpose

このリポジトリは、レベニューアシスタント向け userscript の汎用開発基盤を提供する。

単一機能専用の補助スクリプトではなく、今後複数の画面拡張や情報可視化を段階的に追加できる構成を前提にする。

## Initial Scope

初期スコープは次の通り。

- analyze 画面を対象にする
- 選択日付に対するブッキングカーブ API の取得を安定化する
- 団体室数を月次カレンダー上で見やすくするための PoC へつなげる

## Out Of Scope

初期段階では次を含めない。

- 複数機能の同時実装
- 既存画面全体の大規模な見た目変更
- 外部サーバーや外部保存先との連携

## Initial Public Behavior

初期状態の userscript は、analyze 日付ページを検知し、対象日付の団体室数を取得できる土台を提供する。

この時点では、ユーザー向け UI の変更はまだ行わない。
